SourceTrak Admin API

The IBP SourceTrak API allows one to request dynamic tracking numbers, log page visits and heartbeats via a RESTful service.

Get Number

The default request is made via GET and is used to get a phone number to display on one's site. https://st1.ifbyphone.com/st/

Request Parameters

    
        api=true
        _stk=SOURCETRAK_KEY
        dr=DOCUMENT_REFERER
        dl=DOCUMENT_LOCATION
        dt=DOCUMENT_TITLE
